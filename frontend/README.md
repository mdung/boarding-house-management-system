# Frontend - Boarding House Management System

React application built with Vite and TailwindCSS.

## Requirements

- Node.js 18+
- npm or yarn

## Installation

```bash
npm install
```

## Development

```bash
npm run dev
```

The application will be available at `http://localhost:5173`

## Build for Production

```bash
npm run build
```

The built files will be in the `dist` directory.

## Preview Production Build

```bash
npm run preview
```

## Features

- Admin dashboard with statistics
- Boarding house and room management
- Tenant management
- Contract management
- Invoice generation and tracking
- Payment management
- Tenant portal for viewing invoices and payments

## Default Accounts

- Admin: `admin` / `admin123`
- Tenant: `tenant` / `tenant123`
